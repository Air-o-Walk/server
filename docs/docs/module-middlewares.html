Module: middlewares

Middlewares principales de la API. - CORS: permite solicitudes desde cualquier origen con métodos GET, POST, PUT, DELETE y cabeceras 'Content-Type' y 'Authorization'. - express.json(): parsea el body de las solicitudes con formato JSON. - express.urlencoded(): parsea el body de las solicitudes con formato URL-encoded.
